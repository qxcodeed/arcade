# Retorna se é ou não é

Seja um número inteiro, retorne se ele é primo ou não.

  
## Exemplos

<!-- load tests.toml --tests 3 -->
```py
>>>>>>>> INSERT
1
======== EXPECT
false
<<<<<<<< FINISH
```

```py
>>>>>>>> INSERT
2
======== EXPECT
true
<<<<<<<< FINISH
```

```py
>>>>>>>> INSERT
3
======== EXPECT
true
<<<<<<<< FINISH
```
