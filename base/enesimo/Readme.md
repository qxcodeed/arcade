# L2 - @enesimo - Gere o enesimo primo

## Intro

Dado um número inteiro `n`, gere o `n`-ésimo número primo apenas utilizando recursão.

Utilize o código eh_primo que você já implementou.

O primeiro primo da lista é o número 2.
